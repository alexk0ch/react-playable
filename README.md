# react-playable
React wrapper for [playable](https://github.com/wix/playable)

## Instalation

```
npm install react-playable
```

## Usage

```javascrpt
import ReactPlayable from 'react-playable';

function App() {
  return (
    <ReactPlayable
      width={760}
      height={428}
      title="MY VIDEO!"
      src="http://clips.vorwaerts-gmbh.de/big_buck_bunny.mp4"
    />
  );
}
```
