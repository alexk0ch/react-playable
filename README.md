# react-playable
React wrapper for [playable](https://github.com/wix/playable)

## Instalation

```
npm install react-playable
```

## Usage

```javascrpt
import ReactPlayable from 'react-playable';

functon App() {
  return (
    <ReactPlayable
      width={760}
      height={428}
      title="MY VIDEO!"
      src="http://www.sample-videos.com/video/mp4/720/big_buck_bunny_720p_10mb.mp4"
    />
  );
}
```
